## Pre-requisites
- docker (& docker-compose)
- node js

## How to start
In the Terminal :

    docker-compose up
Visit :  `http://localhost:8080/`

## How to test
In the Terminal :

    npm install
    npx cypress open

## How to run the test
   - Cypress opens automatically
   - Choose Chrome
   - Click "Start 2E2 Testing" in Chrome
   - Browser Chrome open automatically
   - Choose example : "login.cy.js" in the folder "cypress\e2e"
